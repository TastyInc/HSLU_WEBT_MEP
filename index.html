Information

Zentrum dieser Seite ist das Spiel "Blaster". Mit dem Formular unten kann sich der Benutzer einloggen oder registrieren. Beachten sie die Passwortvorgaben bein Registrieren.

Das Spiel wird mit Space gestartet. Ziel des Spiels ist es, möglichst viele Punkte zu erreichen.

Der aktuelle Punktestand wird rechts oben im Canvas angezeigt.

Mit WASD können Sie den Spieler bewegen.

Weichen Sie den roten Gegnern aus und schiessen sie mit der linken Maustaste. Das Zerstören von Gegnern gibt Bonuspunkte.

Sammeln Sie die grünen Boxen ein für Bonuspunkte indem Sie diese berühren.

Das Spiel ist beendet, sobald Sie mit einem Gegner in Kontakt kommen. Danach können Sie das Spiel erneut mit Space starten.

Unterhalb des Canvas werden die Highscores angezeigt. Das Spiel kann auch ohne Registration gespielt, allerdings werden so keine Scores gespeichert.
